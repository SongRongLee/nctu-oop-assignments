# nctu-oop-assignments
Programming assignments for NCTU Object Orientation Programming Class.

**Course No:** 1186  
**Semester:** 1042  
**Teacher:** 黃世強  

## General Key Usage
* **F1~FX**: perform different tasks  
* **1~x**: Adjust parameter in current task  
* **i, I**: ask for input
* **s, S**: show the student information
